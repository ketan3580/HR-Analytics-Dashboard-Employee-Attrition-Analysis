# HR Analytics : Employee Attrition Analysis (Using PowerBI and MS Excel)
## Project Objective
To provide data-driven insights through an interactive Power BI dashboard to support better HR retention strategies.

## Questions (KPIs)

- What is the attrition rate to total count of employee 
- What does the average age indicate about workforce maturity?
- What is the current workforce size?
- How many employees have left the company?
- Average salary of employee that influence attrition of the company ?
- Which Salary Slab & Age group Has Highest Attrition?
- Education background that has highest attrition?
- Compare attrition by gender , Is there any gender imbalance in turnover?
- Highest attrition by department & Job Role
  

- Dashboard Link : <a href="https://github.com/ketan3580/HR-Analytics-Dashboard-Employee-Attrition-Analysis/blob/main/Screenshot%20-%20HR%20Dashboard.png">View Dashboard</a>

## Process
- Verified dataset for missing values, duplicates, and anomalies.

- Cleaned and standardized data types and categorical values.

- Created calculated measures using DAX.

- Built pivot summaries aligned with business KPIs.

- Designed interactive dashboard with slicers and filters.

- Generated insights and validated results.
